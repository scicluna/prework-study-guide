  • The body element represents the visible content shown to the user.
  • Attributes always go in the the first set of carrots.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Using .name allows you to create a new class (in the case class="name" would link to .name in css
  • Using * in css will apply that style to everything on the page
  • Formatting css requires you to first state what type (like h1, h2, etc...) and then in {} type declarations like width : 10px; and the like
